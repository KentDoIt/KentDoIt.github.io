JS 短路求值 Short-circuit evaluation

JavaScript 3 分鐘閱讀時間
2022-06-20

口訣

  • && 左邊是假值(falsy) 則不會執行右邊。
  • || 左邊是真值(truthy) 則不會執行右邊。

下方透過範例（賦予值、表達式），來快速上手 短路求值。

e.g. 使用 || 來實作賦予預設值功能

e.g. && 透過判斷 真假值 來決定是否執行表達式（也可以執行函式）

e.g. 判斷物件屬性是否存在

注意事項

  1. 短路求值 使用表達式的情況下，只適用於沒有 else 的判斷式。
  2. 短路求值 不能帶 return。

詬病「不夠嚴謹」

👨‍💻 當變數為 false、0 或空字串 "" 其中之一時， 則布林值也會轉型為假值 (falsy)，對於判斷變數是否使用過來說是相當不方便的一件事。

javascript 後來增加了一種 空值合併運算子 用來解決不夠嚴謹的問題。
